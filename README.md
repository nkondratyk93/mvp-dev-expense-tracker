# Dev Expense Tracker

Track every SaaS tool, API, and infra cost across your indie projects. Local-first dashboard with project grouping and waste detection. No signup required.

**Live:** https://dev-expense-tracker.visieasy.com

## Features

- Group expenses by project to see per-project monthly costs
- Waste detector — mark unused subscriptions, see total wasted spend
- Zero signup — all data stays in your browser (localStorage)

## Tech Stack

Next.js 15, TypeScript, Tailwind CSS, shadcn/ui, localStorage

## Development

```bash
npm install && npm run dev
```

Open http://localhost:3000
